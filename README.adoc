== What Is This?

WokeLangiser adds **human-centered patterns** to existing code — consent-based
operations, inclusive error messages, and accessibility-first interfaces.

WokeLang is a language designed for human collaboration, empathy, and safety.
WokeLangiser brings these patterns to existing codebases.

== How It Works

Configure patterns in `wokelangiser.toml`. WokeLangiser:

1. Analyses existing error messages and user-facing strings
2. Generates inclusive, accessible alternatives
3. Adds consent patterns (`only if okay`) for destructive operations
4. Adds attribution (`thanks to`) for dependency acknowledgment
5. Generates accessibility metadata (screen reader hints, high-contrast)

== Key Value

* **Inclusive error messages** — clear, respectful, actionable
* **Consent patterns** — destructive operations require explicit agreement
* **Accessibility-first** — screen readers, high contrast, keyboard navigation
* **Empathetic UX** — software that respects the human using it
